# 🍊 AstrBot_Plugin_OrangeVibe (橘子鉴定)

**基于大模型动态生成的“赛博成分鉴定仪”**  
**万物皆可查成分 | LLM辅助制作鉴定 | 动态毒舌锐评 | 互动海报呈现 | 群聊同类雷达**

## 🌟 核心亮点

OrangeVibe (橘子鉴定) 是为 AstrBot 打造的 **LLM 辅助的成分鉴定插件**：

- **🤖 手搓鉴定方案**
  只需要提供一句话（例如：“测测你的群友成分”、“男猫娘接受度调查”），大模型将瞬间化身“成分鉴定架构师”，实时构思生成一份完整的、蕴含多维选择与结果属性分布的专属鉴定方案。
- **✨ “千人千面”毒舌锐评**
  群友做完判定后，系统会将每一步的选择发送回大语言模型。大模型将结合用户答题记录，输出个性化的锐评。
- **🖼️ 全自动鉴定海报**
  测试完毕将自动渲染海报结果，支持自定义海报模板。
- **📡 群聊同类雷达 **
  如果是在群聊中触发的海报渲染，插件还将火速扫描同群的其他成员。如果他们中有人也测试过拿到同款“成分标签”，将会被一并公示于鉴定海报之上，极致的群社交体验！

## 🚀 指令清单

所有常规交互默认使用 `/` 或你系统指定的触发前缀。同时内置了丰富的中文触发别名！
*Note: 指令前缀按照 Astrbot 有关设置确定，如果你的机器人前缀设置为`!`，则指令为 `!vibe 123456`。*

### 日常使用（全员支持）
| 核心指令 | 别称可以直接输入（带前缀） | 说明 |
|:---|:---|:---|
| `/vibe_list` | `/鉴定列表`、`/成分大厅` | 检阅系统已收录并上架的所有现存鉴定方案及其作者署名与简介。 |
| `/vibe_hot` | `/热门鉴定`、`/鉴定排名` | 展示参与度最高、最受欢迎的 Top 5 流行鉴定榜单。 |
| `/vibe [编号]` | `/测成分 [编号]`、`/鉴定 [编号]` | 一键直达，参与该具体编号的指令的成分鉴定。 |
| `/vibe [编号] retry`| `/测成分 [编号] retry`、`/重测成分 [编号]`、`/重新鉴定 [编号]` | 不服AI评价？刷新纪录重新作答相同鉴定！ |
| `/vibe_stop` | `/退出鉴定`、`/停止鉴定`、`/取消` | 随时随地强制中断进行中的测算互动流程。 |
| `/vibe_help` | `/鉴定帮助`、`/测算帮助` | 呼出机器人使用说明及指令清单。 |

### 方案创造与管理
| 核心指令 | 别称可以直接输入（带前缀） | 说明 |
|:---|:---|:---|
| `/vibe_create` | `/制作鉴定`、`/创建鉴定`、`/搓鉴定` | 进入引导式创建流程，告诉大模型脑洞主题和要求，1分钟自配一套专业鉴定方案！ |
| `/vibe_del [编号]` | `/删除鉴定 [编号]` | 销毁对应的鉴定方案库（仅限该方案制作者，或 Bot 管理员）。 |

## ⚙️ 部署说明与环境依赖

1. 确保已正确配置 AstrBot 的 T2I 功能，建议自行部署。

2. 在 AstrBot `data/plugins/` 目录下克隆本项目（重命名为 `astrbot_plugin_orangevibe`）：
   ```bash
   git clone https://github.com/largefox/astrbot_plugin_orangevibe.git astrbot_plugin_orangevibe
   ```
   或直接下载本项目zip包，在插件页上传即可。
3. 配置一个强大的聊天服务商（建议选智商高、不轻易带 AI 语气词的发散型模型以得到最佳体验）。

## 🔧 后台配置面板

通过 AstrBot 可视化 Web 控制台即可找到 `橘子鉴定`，修改即时生效：

### 常规限制配置
| 配置项 | 默认值 | 说明 |
|:---|:---|:---|
| **允许在群聊中互动答题** | `false` | `allow_group_vibe`。✅ 开启: 不受限制。<br>❌ 关闭: 群内测算会被提示转移防刷屏。但已做完的测算生成的成就海报仍可在群聊发送。 |
| **允许在群聊中制作鉴定** | `false` | `allow_group_create`。✅ 开启: 不受限制。<br>❌ 关闭: 保护群内文字空间，要求在私聊环境与 AI 对话制作新方案。 |
| **仅限管理员新增鉴定方案** | `false` | `admin_only_create`。✅ 开启: 只有管理员出题并进库，禁止普通水友操作，避免数据过载。 |
| **每日制作鉴定方案次数上限** | `3` | `daily_create_limit`。每个普通用户每天能够挥霍算力生成新鉴定方案的数量。（Bot 管理员无视此限制）。 |
| **每次制作方案的修改次数上限** | `8` | `max_modify_count`。在生成一份新鉴定草稿后，最多允许与 AI 发生多少次退回修改及重制要求对话。 |

### 海报模板文案
| 配置项 | 默认值 | 说明 |
|:---|:---|:---|
| **海报说明文案** | `对bot发送以上带有编号...` | `invite_poster_tip`。展示在新方案发布海报中间及结果海报下方的行为引导语。<br>可用 `{test_id}` 动态嵌入号码。 |
| **海报底部水印文字** | `使用 Astrbot 插件...` | `footer_text`。自定义展示在所有发出的鉴定海报最底部的引流或版权注释。 |
| **结果海报主题** | `default` | `result_theme`。在插件目录下的 `templates/` 系统中选用自己编写的结果海报前端渲染逻辑皮肤。 |
| **分享海报主题** | `default` | `invite_theme`。在插件目录下的 `templates/` 系统中选用自己编写的分享海报皮肤。 |

### 🎨 定制化功能：自制打分图与海报皮肤！
支持完全自定义结果海报与分享图模板（Jinja2格式）：
- 前往插件目录 `templates/`
- 自行创建 `result_自定主题名.html` 和 `invite_自定主题名.html`
- 分别在控制台设置中配上您的 `自定主题名`，完美兼容你的视觉要求！
